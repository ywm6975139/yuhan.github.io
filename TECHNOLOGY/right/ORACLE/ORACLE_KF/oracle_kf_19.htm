索引

如果指定了PRIMARY KEY或者UNIQUE约束，那么将自动创建索引。

 

用户可以在某个列上建立非唯一的索引，以加快基于该行的查询.

 

创建索引

CREATE INDEX   emp_ename_idx

ON   emp(ename);

 

什么时候创建索引？

–欲创建索引的列在 WHERE子句或者连接条件中频繁使用.

–该列所包含的不同值很多.

–该列包含大量的空值.

–表中的数据行数非常大，而且只有 2–4% 数据行被查询出来.

什么时候不必创建索引？

–表是空的.

–列在查询条件中不经常使用.

–大多数基于该表的查询，所查询出的数据量远多于2–4% 行.

       -表被频繁修改
