能够创建和管理用户

创建组

把用户加入到组

把普通用户变成ubuntu管理员

使用自己创建的管理员 创建普通用户 sudo

 

useradd -m -d /home/stu01 stu01   创建用户和主目录，不指定密码

useradd -u 588 -m -d /home/stu01 stu01  可以选择性地指定用户的ID

useradd -G admin -m -d /home/stu01 stu01 创建时就添加到管理员组

passwd stu01 修改密码

userdel -rf stu01 删除用户和它所在的目录

/etc/shadow 用于保存密码 账号有效期等信息  只有root能够访问

 

groupadd javaclass 创建组

 

gpasswd -a stu01 javaclass 将用户添加到组

gpasswd -d stu01 javaclass 将用户从组里删除

 

gpasswd -a stu01 admin 变成管理员了 或者改/etc/sudoers文件？

 

useradd yuhan 使用自己建的管理创建新用户

计划任务

   创建周期性 计划

crontab -e 进入任务计划文件编辑

 

前五个分别表示 分钟 小时 日期 月份 星期

使用“*”表示任何时间

使用减号“-”可以表示一个连续的时间范围，如“1-4”表示整数1,2,3,4

使用逗号“,”可以表示一个间隔的不连续范围，如“3, 4, 6, 8”

斜杠符号“/”可以用来指定间隔频率，如在日期字段中的“*/3”表示每隔3天

 

crontab -l 查看用户计划

crontab -r 删除用户的任务计划

第22章 Ubuntu安全性

 

能够设置ubuntu网络安全

关掉所有的口，只开80和22，22端口只允许特定的人能访问

 

apt-get install iptables 安装防火墙

iptables -l 查看所有的策略

iptables -p input drop 配置默认拒绝所有的流量

iptables -t filter -A INPUT -p TCP --dport 80 -j ACCEPT    只开80端口



iptables -t filter -A INPUT -s 192.168.80.0/24 -p TCP --dport 22 -j ACCEPT 

只允许 192.168.80.0 255.255.255.0这个网段的人访问22

 

iptables-save >/etc/iptables.up.rules 保存配置
