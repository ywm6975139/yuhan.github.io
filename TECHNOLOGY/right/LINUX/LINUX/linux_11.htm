ifconfig 查看活动着的网口

 

ifconfig -a 可以查看到不活动的网口

 

ifdown eth0 禁用eth0网口

ifup eth0  启用eth0网口（并重加加载配置文件/etc/network/interfaces）

 

ifconfig eth0 down 禁用eth0网口

ifconfig eth0 up 激活eth0网口（不加载配置文件）

 

ifconfig eth0 查看指定的网口

 

ping -c 2 -s 1024 10.7.10.1 发送指定个数，指定大小的数据包

 

route -n 查看路由表

 

vi  /etc/network/interfaces 设置静态ip地址，子网掩码，网关

 

/etc/init.d/networking restart   重启网络

 

apt-get install traceroute  安装traceroute工具

 

traceroute 8.8.8.8 查看icmp包经过的路径

 

netstat -anpt 以数字形式显示系统中所有的TCP连接的信息

 

ifconfig eth0 10.7.10.1 netmask 255.255.255.0 暂时设置一下ip和子网掩码

ifconfig eth0 10.7.10.1/24  这么写也行

 

ifconfig eth0:0 10.7.10.1

ifconfig eth0:1 10.7.10.2   为一个网口添加多个IP地址

 

ifconfig eth0:0   查看子网口的ip地址

 

ifconfig eth0 down

ifconfig eth0 hw ether AA:BB:CC:DD:EE:FF  暂时修改mac地址

ifconfig eth0 up

 

以下两个方法为永久修改mac地址

方法一：

    1）编辑“/etc/init.d/rc.local”文件（sudo gedit /etc/init.d/rc.local）

    2）在此配置文件的最后面加上如（Ubuntu：修改网卡的MAC地址）的修改命令：
         ## 修改 eth0 的 MAC 地址
         sudo ifconfig eth0 down
         sudo ifconfig eth0 hw ether AA:BB:CC:DD:EE:FF
         sudo ifconfig eth0 up

 

方法二：

    1）编辑“/etc/network/interfaces”文件（sudo gedit /etc/network/interfaces）

    2）在此配置文件中找到“iface eth0 inet static”行，在其后面加上“hwaddress ether AA:BB:CC:DD:EE:FF”，如何没有这一行，则自己手动完整添加

PS：其实有的时候方法二并不可行，而方法一有时也会一开机并不起作用，此时就要关闭网络连接，然后再打开，那个方法一的配置文件就生效了。

 

route add -net 10.7.1.0/24 gw 10.7.10.1 添加到指定网段的路由记录

 

route del -net 10.7.1.0/24 删除指定网段的路由记录

 

route add default gw 10.7.10.1 添加默认网关

route del default gw 10.7.10.1 删除默认网关

 

hostname yuhan  设置主机名字为yuhan，暂时的

 

arp -a 查看arp解析的网关的mac地址

 

arp -s 【ip地址】 【mac地址】         绑定固定的mac地址

 

vi /etc/hostname 主机名称配置文件

 

vi /etc/resolv.conf 域名服务器配置文件

 

nslookup 测试域名解析

 

vi /etc/hosts 编辑hosts文件
