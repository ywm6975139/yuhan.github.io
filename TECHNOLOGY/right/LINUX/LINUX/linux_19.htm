装

apt-get install vsftpd

 

默认不允许匿名用户登录，修改配置文件可以允许匿名用户登录

 

 

匿名用户默认使用ftp账户，默认的ftp目录为/srv/ftp

 

修改匿名用户的默认目录

mkdir -p /srv/files/ftp/

chmod -R 755 /srv/files/ftp/   不能改成777，如果改成777的话，不让匿名用户登录

usermod -d /srv/files/ftp ftp    这目录只能读不能写

 

让匿名用户也可以写的目录

mkdir /srv/files/ftp/upload/

chmod 777 /srv/files/ftp/upload/

重启ftp服务，ok！

 

 

FTP的主动模式和被动模式

主动模式：就是端口不可以改变，就是20和21端口

被动模式：就是端口可以给，只要大于1024，可以随便给

 

要改的话得从ftp服务器端和客户端都要改

先说服务器：

重启ftp。这样一改，服务器端就可以支持被动模式

 

再说客户端

internet属性->高级   里面可以改，这个不用说太多了吧
