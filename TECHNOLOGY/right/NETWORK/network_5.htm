路由器

 

enable 进入特权模式

 

config terminal 进入全局配置模式

 

show ip protocol  看看用的是啥路由协议

 

show ip interface brief  看看每个网口的详细信息

 

show ip interface fastEthernet 0/0 查看f0/0端口的详细信息

 

show flash 查看路由的操作系统

 

show version 查看操作系统的版本

 

hostname yuhan   改名字

 

(config)#line vty 0 4

(config-line)#password 123456   设置telnet密码

 

(config)#line console 0

 

(config-line)#password 123456 设置控制台密码

 

(config)#enable password 123456 设置特权密码

 

(config)#enable secret 123456 设置加密的特权密码，不加密的失效

 

ip address 192.168.80.100 255.255.255.0 设置ip地址和子网掩码

 

show controllers serial 2/0 查看串口是否DCE

信息传输的方式：

DTE<====>DCE<===>网络<===>DCE<====>DTE

换句话说DCE是终端设备，比如计算机，用于人机交互等；DTE是接口设备，比如MODEM，负责将信息接入网内。

clock rate 64000 配置DCE的时钟频率

 

copy running-config startup-config  保存当前配置到NVRAM

 

ip route 172.16.1.0 255.255.255.0 172.16.3.2  添加静态路由

 

show ip route 显示路由表

 

arp -a 显示IP到mac地址的映射（windows上使用）

 

show ip arp 显示IP到mac地址的映射（cisco设备上使用）

 

ip route 0.0.0.0 0.0.0.0 172.16.3.2 设置默认路由

 

ip route 172.16.0.0 255.255.248.0 192.168.1.1 路由汇总，就是构造超网的意思

 

RIP路由协议（基于距离向量的路由选择协议）

Router(config)#router rip

Router(config-router)#network 192.168.0.0       

Router(config-router)#network 192.168.1.0

Router(config-router)#version 2             

Router(config-router)#no auto-summary 当遇到不连续子网则需要手动关闭路由汇总

 

eigrp路由协议

Router(config)#route eigrp 10

Router(config-router)#network 192.168.0.0

Router(config-router)#network 192.168.1.0

Router(config-router)#no auto-summary 遇到不连续子网则也需要手动关闭路由汇总

show ip rigrp topology  显示最佳和备用路径 

RIP的优先级比eigrp的低，有啥冲突，RIP还得听eigrp的，首先eigrp收敛快，算出来的路径也是最佳的，就算最佳的突然down了，备用路由也可立即启用

 

OSPF路由协议（示例为单区域）

Router(config)#router ospf 1

Router(config-router)#network 192.168.0.0 0.0.0.3 area 0

Router(config-router)#network 172.16.0.0 0.0.255.255 area 0

 

show ip ospf neighbor 看看邻居

show ip ospf database 看看数据

 

tracert 跟踪路由(在window计算机上使用，发的是icmp包)

traceroute 跟踪路由（在linux、BSD、router下使用，发的是UDP包）

 

ip subnet-zero     允许你在自己的网络设计和使用第一个和最后一个子网。

 

 

交换机

show mac-address-table 查看交换机的mac地址表（刚开始的时候，看不见，因为交换机不知道哪个端口对应着哪个mac，得ping一下才看得到）

 

spanning-tree vlan 1 priority 4096  将优先级提高（生成树协议）

 

实现端口安全

interface fastEthernet 0/4       先进去交换机0/4的接口

switchport mode access    配置该接口为访问接口

switchport port-security  设置安全策略

switchport violation shutdown  违反安全策略端口关闭

---------

switchport port-security maximum 1  最大连接数为1

switchport port-security mac-address 0090.0CD7.65C8 绑定mac地址

......

 

VLAN

默认就是在vlan1里面

vlan 2    创建 vlan2

no vlan 2   删除vlan2

---------

interface fastEthernet 0/2 进去一个接口

interface range fastEthernet 0/2-5      进去多个接口

---------

switchport access  vlan 2    将端口指定到vlan2

 

独臂路由

interface gigabitEthernet 6/0

no shutdown

exit

interface gigabitEthernet 6/0.1   进入子接口

encapsulation dot1Q 1  指定该接口作为 VLAN1 的网关

ip address 192.168.0.1 255.255.255.0

no shutdown

exit

interface gigabitEthernet 6/0.2

encapsulation dot1Q 2

ip address 192.168.1.1 255.255.255.0

no shutdown

------------

switchport mode trunk   在交互机连接路由器的接口开启trunk

使用ACL保护路由器安全

show access-lists 查看访问控制列表

access-list 101 permit tcp192.168.2.0 0.0.0.255host 10.0.0.2 eq 80

                                                                                   host 10.0.0.2=10.0.0.2 0.0.0.0

access-list 101 permit ip 192.168.2.0 0.0.0.255 192.168.0.0 0.0.0.255

access-list 101 permit any

                                       any=0.0.0.0 255.255.255.255任何一个地址都满足

 

ip access-group 101 out/in

还可将访问控制列表加载到vty端口

 

 

NAT地址转换

ip nat inside source static 10.1.1.1 172.16.12.10   指定静态对应关系

--------------------------------

Router(config)#ip nat pool todd 172.16.12.10 172.16.12.14 netmask 255.255.255.0 

Router(config)#access-list 1 permit 10.1.1.0 0.0.0.255 

Router(config)#ip nat inside source list 1 pool todd  （overload）指定动态对应关系

--------------------------------                                如果是PAT就加一个overload

Router(config)#interface fastEthernet 0/1

Router(config-if)#ip nat inside    指定内网接口

Router(config)#interface s0/0

Router(config-if)#ip nat outside   指定外网接口

 

debug ip nat 显示地址转换信息

 

IPV6的设置

ipv6 unicast-routing   启用IPV6

ipv6 address 2001::1/64   配置ipv6地址,2001是网段，1是主机号，/64是子网掩码
