ok，现在jboss已经安装成功，我们需要把jboss和apache结合一起一起为我们服务。

 

 

 

JBoss 与 Apache 集成：

1.安装 Apache Web 服务器。

三个空随便写，没关系

一直下一步，然后就安装完成了。

 

2.导航到 <Apache 主目录>\conf 目录。

 

3.导航到 ApacheIntegration\windows，它位于 ALM Platform DVD （就是ALM11的安装文件）上。将显示 qc_integration 文件夹。

 

4.将 qc_integration 文件夹和其内容复制到 <Apache 主目录>\conf 目录。

 

5.将 mod_jk.so 复制到 <Apache 主目录>\conf\qc_integration 目录。

 

6.在 <Apache 主目录>\conf 目录中，将以下行添加到 httpd.conf 文件：

#ALM Platform integration:This includes points to the Jboss-Apache #configuration

file (for ALM Platform)

#Load mod_jk module

LoadModule jk_module conf\qc_integration\mod_jk.so

Include conf\qc_integration\mod_jk.conf

 

7.重新启动 Apache Web 服务器。

现在可以使用以下 URL 连接到 ALM Platform 站点：

http://<ALM Platform 服务器名称>[<:端口号>]/qcbin。
